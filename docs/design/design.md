# 디자인 초안

이 파일은 프로젝트의 기본 디자인 가이드와 컴포넌트 목록을 담습니다.

## 목적
- 화면별 레이아웃과 공통 컴포넌트 지침 제공

## 권장 컴포넌트
- Navbar, Footer
- PostCard, ProductCard
- AuthForm, ProtectedRoute

## 현재 구현된 컴포넌트
- `Navbar` : 데스크탑 상단 메뉴 + 모바일 드로워
- `HomePage` : 오렌지 히어로, 로그인 카드
- `BoardPage` : 게시글 목록, 글쓰기 폼, 댓글, 좋아요
- `SectionPage` : 섹션별 임시 페이지

## 반응형 가이드
- 모바일 우선 디자인
- Tailwind 유틸리티 기반 클래스 사용

## 컬러 및 타이포그래피 (초안)
- 기본 색상: primary / neutral / accent
- 폰트: 시스템 폰트 스택 또는 지정된 웹폰트

## 레이아웃
- 컨테이너 최대 너비 1200px
- 가로 패딩 16px, 섹션 간격 24px

## 참고
- 디자인 확정 후 컴포넌트화 작업 권장
